# Stackflow 시작하기

**Stackflow**는 모바일 디바이스(iOS/Android 등)에서 주로 활용되는 Stack Navigation UX를 JavaScript 환경에서 구현하기 위한 프로젝트에요.

- 화면을 쌓고 스크롤을 유지해요.
- 화면이 쌓이는 전환효과와 뒤로가기 시 화면이 사라지는 전환효과를 지원해요.
- iOS 백스와이프 구현에 대한 레퍼런스를 제공해요.
- 다음 화면에 파라미터를 전달해요.

그렇다면 Stackflow는 기존 Navigation 라이브러리와 대비해서, 어떤 장점을 가지고 있을까요?

- UI 없이 Stack 상태만 사용 가능해요. 원하는 입맛대로 UI를 뜯어서 사용할 수 있어요.
- 플러그인 인터페이스를 통해, 원하는 기능을 Life Cycle 사이에 주입가능해요.
- 코어 로직과 통합 레이어가 분리되어 있어서, 다양한 플랫폼에 적용 가능해요.
- 렌더 로직을 외부에서 주입 받으므로, 데스크탑 지원이 용이해요.
- Server-Side Rendering을 지원해요. (`ReactDOMServer.renderToString`)

<Callout emoji="💡">
  2022년 6월 시점에서는 React와 React DOM까지 레퍼런스로 지원하고 있어요.
</Callout>

- 본격적으로 Stackflow를 사용하기 위한 [사전 준비사항](./prerequisites)을 알아보고 [설치 및 설정하기](./installation-and-setup) 문서를 확인해요.
